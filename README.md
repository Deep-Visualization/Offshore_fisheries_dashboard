# Offshore Fisheries Analytics Dashboard

An interactive data analytics dashboard built using **Streamlit, Pandas, and Plotly** to explore U.S. commercial offshore fisheries landings data.

This project transforms raw fisheries data into an intuitive, interactive visualization tool that allows users to analyze production trends, compare states, and explore geographic distribution.

---

# Features

- Interactive state selection (Add / Remove / Select All / Clear All)
- Year range filtering
- Metric toggle (Pounds vs Dollars)
- Dynamic KPI cards (Total, Max Year, Min Year)
- Time-series trend analysis
- Top 10 states bar visualization
- Choropleth map for geographic distribution
- Download filtered dataset as CSV
- Animated ocean-themed UI

---

# What This Dashboard Shows

This dashboard enables users to:

- Analyze how many pounds of fish were landed offshore each year
- Compare total revenue generated across states
- Identify top-performing states
- Visualize trends over time
- Explore geographic production patterns across the U.S.
